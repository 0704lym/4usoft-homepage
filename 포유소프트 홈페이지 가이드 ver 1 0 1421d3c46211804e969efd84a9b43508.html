- scss 파일/폴더규칙

  • 컴파일이 필요 없는 파일은 _를 맨 앞에 포함하여 작성.
  • 가능하면 컴포넌트 형식으로 분할하여 작성하고 하나의 스타일시트에 임포트 하는 방식으로 구성한다.
  • scss 코드 작성 규칙
    1. 속성
    • 모든 속성은 영문 소문자로 작성한다.
    • 약식으로 가능하면 약식으로 작성한다.
    • 컬러값은 최대한 단축값으로 작성한다. (ex. #fffff → #fff)
    .class-name {
  background: #fff url("../images/image.jpg") no-repeat 50% / cover;
}
    • 기본요소에 box-sizing 속성값 작성하지 않는다.
      /* X */
* {
  box-sizing: border-box;
}

/* O */
.class-name {
  box-sizing: border-box;
}
    1. 선택자 및 우선 순위
    • 공통선택자(*)는 웹페이지 성능저하를 가져오므로 사용하지 않는다.
    • 태그를 선택자로 스타일을 작성하는 것은 지양한다.
    • !important 는 불가피한 상황에만 사용한다.
    • 인라인 스타일의 우선순위가 높으므로 인라인 스타일의 사용은 최소한으로 한다.

    1. 세미 콜론 : 속성마다 세미콜론을 사용한다.
    1. 들여쓰기 : 1탭 들여쓰기를 하며 1탭의 크기는 공백 2칸으로 설정한다.
    1. 공백의 사용 : 선택자 간, 중괄호 간 한칸의 공백을 사용. 속성 선언 시 ‘:’뒤에 한 칸의 공백을 사용한다.
      .class-name {
    color: red;
}
    1. 줄바꿈 : 중괄호 사이와 속성 작성시 줄바꿈 처리하며 중첩시에 선택자 전에 줄바꿈 처리한다.

    (단, after,before 와 같이 가상클래스는 줄바꿈 처리하지 않는다.)

    ul.card-content {
  padding: 30px;
  border-radius: 16px;
  background-color: rgba(255, 255, 255, 0.6);
  min-height: 323px;
  box-sizing: border-box;

  li {
    line-height: 28.8px;
    margin-bottom: 10px;
    &:before {
      content: "";
      display: inline-block;
      margin: 0 12px;
      width: 4px;
      height: 4px;
      background-color: $basic;
      border-radius: 50%;
      vertical-align: middle;
    }
  }
}

    1. 주석
    • 주석 작성 시 종료 주석은 작성하지 않는다.
    • scss 중첩 시 알아볼 수 있도록 주석으로 클래스를 작성한다.
    .header {
  &-menu {
    ...;
  } // .header-menu
}

    8.반복

    • 반복적으로 사용되는 스타일의 경우 mixindlsk scss함수로 활용한다.
    • 반복적으로 사용되는 값은 변수로 선언하여 활영한다.
    • mixin 이나 변수, 함수명 등을 작성할 때는 카멜형식으로 표기한다.
    • 스타일이 연속으로 반복되는 경우 for문 사용하여 추가 및 삭제에도 수정가능하도록 한다.
    @for $i from 1 to 5 {
  .tab-icon-0#{$i} {
    display: inline-block;
    width: 100rem;
    height: 100rem;
    background-image: url("/asset/image/pc/tab_icon_0#{$i}.png");
    background-size: contain;
    background-position: center;
    background-repeat: no-repeat;
    @media (min-width: $breakpoint-mobile) and (max-width: $breakpoint-tablet) {
      @include cardStyle(60px, 60px);
      background-image: url("/asset/image/pc/tab_icon_0#{i}.png");
    }
  }
}

- js 파일 규칙

  • jquery로 작성
  • 함수 선언 후 $(function() { } 안에 호출 하는 방식으로 작성함.
  • $(function() { } 안에 바로 함수명 없이 사용하지 않는다.
  • 함수선언 시, 어떤 기능을 위한 코드인지 간략하게 주석으로 메모함.
//다국어 드롭다운
function langdrop() {
  const dropBtn = $('.language-selector button');
  const dropdown = $('.language-selector');

  dropBtn.on('click', function () {
    dropdown.toggleClass('active');
  });

  $(document).on('click', function (event) {
    if (!$(event.target).closest('.language-selector').length) {
      dropdown.removeClass('active');
    };
  });
};
